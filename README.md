## 📌 Proje Hakkında
OneTech; kurumsal ve bireysel kullanıcıların ChatGPT, Gemini gibi üretken yapay zeka araçlarını kullanırken hassas verilerini (TC Kimlik, E-posta, Telefon, Kişi/Kurum isimleri ve Özel Kurallar) dışarı sızdırmasını önleyen akıllı bir tarayıcı eklentisidir.

Kullanıcı prompt gönderdiği anda araya girer, Doğal Dil İşleme (NLP) altyapısıyla metni anlık olarak analiz eder. Hassas veri tespit edildiğinde gönderim işlemini durdurur ve kullanıcıya interaktif bir güvenlik kalkanı sunar.

## 🛠️ Teknolojik Altyapı
- **Frontend:** JavaScript / Web Extensions API (Chrome Eklentisi)
- **Backend:** Node.js & Express (Render.com üzerinde canlı bulut altyapısı)
- **Doğal Dil İşleme (NLP):** Compromise Kütüphanesi
- **Yönetim Paneli:** HTML5, Tailwind CSS, Chart.js (GitHub Pages entegreli)

## 🚀 Özellikler
- **Anlık NLP Analizi:** Prompt içindeki isimleri ve şirketleri yapay zekaya gitmeden yakalar.
- **İnteraktif Güvenlik Kalkanı:** Kullanıcı maskelenen verileri anlık olarak açıp kapatabilir, risk analizini görebilir.
- **Canlı Dashboard:** Engellenen veri istatistikleri grafiklerle takip edilebilir.
- **Dinamik Kural Yönetimi:** Panel üzerinden sisteme anında yeni yasaklı kelimeler (şirket sırları) eklenebilir veya silinebilir.
